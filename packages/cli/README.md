# mstack

Install Misbah Khursheed's Build Like This product-to-production engineering workflow in any repository.

```sh
npx mstack@latest
```

Then configure the AI coding environments you use:

```sh
mstack ai setup
mstack status
```

mstack previews changes, preserves user-owned documents, records generated-file ownership, and supports Claude Code, OpenAI Codex, Cursor, Gemini CLI, Continue, and Aider.

See the complete [CLI guide](https://github.com/misbah-khursheed/buildlikethis/tree/main/docs/cli) for commands, configuration, AI runtime details, troubleshooting, and migrations.
